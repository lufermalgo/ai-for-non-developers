# SESSION_STRATEGY.md

## Continue Same Session When

- same feature
- same bug
- same module
- same decision chain

## Start New Session When

- new project
- unrelated task
- noisy conversation
- architecture changed completely
- context feels bloated

## Before Reset

Ask for:

Summarize current decisions so we can continue in a new session.

## Goal

Preserve useful context.
Discard waste.
